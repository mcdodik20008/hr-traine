# RAG Real-Time Coach Integration - Quick Start

После установки зависимостей, инициализируйте RAG систему:

```bash
python -m app.scripts.initialize_rag
```

Это создаст FAISS индекс с базой знаний (53 документа).

Бот автоматически загрузит RAG Coach при первом интервью.

## Проверка работы

Запустите интервью и попробуйте вопросы:
- "Сколько вам лет?" → предупреждение о ТК РФ
- "Вы умеете работать в команде?" → подсказка об открытых вопросах
- "Расскажите о проекте" → одобрение техники STAR
